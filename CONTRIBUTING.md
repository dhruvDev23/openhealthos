# Contributing to ImmuCore

Thank you for your interest in contributing to ImmuCore! We are building a local-first, privacy-respecting personal health operating system, and we welcome contributions of all kinds.

## How to Contribute

### 1. Reporting Bugs
- Check the Issues tab to see if the bug has already been reported.
- If not, open a new issue using our Bug Report template.

### 2. Suggesting Enhancements
- Open an issue explaining the feature you would like to see and why it adds value to the project.

### 3. Pull Requests (PRs)
1. Fork the repository.
2. Create a new branch for your feature or fix (`git checkout -b feature/amazing-feature`).
3. Ensure your code passes all unit tests.
4. Commit your changes (`git commit -m 'Add some amazing feature'`).
5. Push to the branch (`git push origin feature/amazing-feature`).
6. Open a Pull Request against the `main` branch.

## Code of Conduct
Please be respectful and constructive in all community interactions.
